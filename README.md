# 📝 Registration Form

A simple and responsive **Registration Form** built using **HTML** and **CSS**. This project marks my **first step into Front-End Development** and helped me understand the fundamentals of web design, form creation, and CSS styling.

## 🚀 Features

* Clean and modern user interface
* Responsive design
* User-friendly registration form

* Custom CSS styling

* CSS3
